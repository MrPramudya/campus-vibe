🌐 CampusVibe - Semarang Student Hub

CampusVibe adalah platform kurasi ruang produktif (Work From Cafe) dan tempat berkumpul (Hangout) yang dirancang khusus untuk kebutuhan mahasiswa di Kota Semarang.

🔗 Live Demo

Akses aplikasi secara langsung di: https://campus-vibe-marcellus.vercel.app/

🚀 Fitur Unggulan

Campus-Centric Logic: Filter pencarian berdasarkan radius terdekat dari kampus utama (UNDIP, UNNES, UDINUS, UNIKA, UIN, USM).

Dual Mode UI: Desain responsif dengan dukungan Dark Mode dan Light Mode yang modern.

Vibe Categories: - WFC Mode: Fokus pada koneksi Wi-Fi kencang, ketersediaan colokan, dan suasana tenang.

Hangout Mode: Fokus pada estetika tempat, kenyamanan bersosialisasi, dan jam operasional.

VibeAI Assistant: Simulasi asisten cerdas yang memberikan rekomendasi personal untuk mahasiswa.

Direct Navigation: Integrasi satu klik ke Google Maps untuk rute perjalanan.

🛠️ Teknologi yang Digunakan

Core: React.js (Vite)

Styling: Tailwind CSS (Modern UI/UX)

Icons: Lucide React

Infrastructure: Vercel for Deployment, GitHub for Version Control.
